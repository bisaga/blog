#### Add Nuget library to project

dotnet add \[PROJECT\_NAME\] package \[NUGET\_PACKAGE\] 

**example:**
dotnet add Bisaga.Web package Newtonsoft.Json

#### Add project to solution

dotnet sln \[SOLUTION\_NAME\] add \[PROJECT\_NAME\]
